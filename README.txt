Project 1 - Lunar Lander
CSCI 312 Winter 2021
By Harry Pinkerton
Required to run: Dr. Racket version 7.9


==============
lunar.scm
==============
Allows the user to play a lunar lander game with different strategies to successfully land the spacecraft. Initial height is set to 50 km, initial fuel is set to 20, and initial  velocity is set to 0 km/sec. The spacecraft crashes if it is moving at 0.5 km/sec or faster after reaching the ground.
